# Lesson 9 — Wild Files: Meet the Meerkat 🦫

The first lesson of a new **information-text** unit for Grade 5 English.
The child reads a real article about meerkats in the Kalahari, then mines
it for grammar — **the perfect tenses** (present & past perfect) and
**conjunctions & clauses** (main, subordinate & relative). Every file is
**fully self-contained** (fonts, images and styles are all baked in) —
there are no folders, dependencies or build steps.

Kalahari-safari themed throughout, with real meerkat photos, cartoon
meerkat characters, an Africa map highlighting the three Kalahari
countries, and three break-time games. British English throughout.

## What's inside the lesson
- **Read to Learn** — the "Meet the Meerkat" article, split by subheadings,
  with ⚡ Field Challenges that preview each grammar skill.
- **Where in the world?** — an Africa map marking Botswana, Namibia and
  South Africa.
- **Comprehension** — five full-sentence questions (facts + text features).
- **The Perfect Tenses** — teach + practice (present perfect, past perfect,
  fixing tricky past participles).
- **Conjunctions & Clauses** — teach + practice (coordinating,
  subordinating & relative; the fronted-clause comma).
- **Write Your Own Fact File** — a honey-badger writing task with a checklist.
- **Homework & Exit Ticket**, then a games send-off.

## Files
| File | What it is |
|------|------------|
| `index.html` | The lesson slides (open this first) |
| `feed-the-meerkat.html` | "Feed the Meerkat" — a word-guessing game (hangman, re-themed) |
| `kalahari-connect-4.html` | "Kalahari Connect 4" — meerkat vs the Wild, vs the computer |
| `kalahari-word-search.html` | "Kalahari Word Search" — find 14 hidden desert words |

The lesson's final slide links to the three games; the links use these
exact filenames, so keep them as they are. Each game links back to
`index.html`.

## How to run the lesson
1. Open `index.html` and share your screen.
2. Read the article aloud together, pausing at each ⚡ Field Challenge.
3. Work through the grammar and writing tasks; the child answers aloud or
   on paper. (Tutor answers live in the separate lesson document, not on
   the pupil slides.)
4. Finish with a break-time game.

## Host it on GitHub Pages
1. Create a new GitHub repository.
2. Upload these `.html` files (and this README) to it.
3. Go to **Settings → Pages**, set **Source: Deploy from a branch**,
   pick the `main` branch and the `/ (root)` folder, then **Save**.
4. After a minute your lesson is live at
   `https://<your-username>.github.io/<repo-name>/`

That's it — no other set-up needed.
